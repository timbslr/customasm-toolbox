# Change Log

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.3.1]

- added option to highlight clobbered registers
- removed option to configure the extension inside the package.json file. A .customasm.json file is now always required!

## [0.2.2]

- static syntax highlighting works again

## [0.2.1]

- mnemonics inside comments are no longer highlighted

## [0.2.0]

- Operand completions are now implemented: As you type your mnemonic followed by operands, the extension will give you suggestions what operand to insert next based on previous operands, the current operands rule definition and typed letters

## [0.1.1]

- Added hovering support for mnemonics that have multiple definitions

## [0.1.0]

- Initial release
